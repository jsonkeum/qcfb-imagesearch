Image Search API

by QuanchiFootball


This is a simple API that will take a search query and generate a list of images from the web.


Example

https://qcfb-imagesearch.glitch.me/api/dogs

Returns an array of ten objects containing the following:

{
"url":"http://cdn3-www.dogtime.com/assets/uploads/gallery/goldador-dog-breed-pictures/puppy-1.jpg",
"snippet":"Goldador Dog Breed Information, Pictures, Characteristics & Facts ...",
"thumbnail":"https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcQMnUZX0tOX2N4907DAsF__q78__Fgb6PqCB1dQHv_ghTWNfw4DsEtMvH8",
"context":"http://dogtime.com/dog-breeds/goldador"
}

API can also take an offset query in the form of an integer value.

https://qcfb-imagesearch.glitch.me/api/how to bake pizza?offset=10


History

https://qcfb-imagesearch.glitch.me/history

Returns an array of maximum ten objects containing prior history records:

{
"term":"dogs",
"time":"Sat Aug 12 2017 05:05:26 GMT+0000 (UTC)"
}
